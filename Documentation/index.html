Welcome to the DB2 Health Monitor Sample Application for PHP

SOAP Edition - Version 1.0

The DB2 Health Monitor Sample Application provides a demonstration as to how Web services can be built using PHP and consumed using PHP and AJAX. All files and directories in the sample are organized and named so that you can understand what is necessary to create a PHP SOAP-based Service Oriented Architecture application.

Although the sample is designed for a multi-tier environment, it can be deployed on a single, physical machine. It includes setup instructions, architectural diagrams, file-level explanations, code documentation, and JavaScript widget documentation for reusing components in other Web applications.

Purpose

The DB2 Health Monitor Sample for PHP illustrates how to write a PHP Web service that serves and consumes health monitoring data from a DB2 database. It includes examples of how to perform the following tasks:

  • Create a PHP SOAP-based Web service using the native PHP 5 SOAP extension.
  • Use an existing WSDL document with SOAP.
  • Extract data as an XML document from DB2 using a single XQuery statement (pureXML™).
  • Validate user and XML inputs using regular expressions.
  • Consume a PHP Web service using the WSDL document and AJAX techniques.
  • Preserve user session information throughout AJAX updates.
  • Create a set of AJAX-enabled, object-oriented, JavaScript widgets.
