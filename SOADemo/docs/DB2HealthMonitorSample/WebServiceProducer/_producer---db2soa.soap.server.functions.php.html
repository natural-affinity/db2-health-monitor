Web Service Function to obtain ALL Health Indicators/Alerts without history as XML

string GET_ALL_DETAILED_INFO (string $username, string $password, string $database, string $port, string $language, boolean $defset)
  • string $username: the username used to connect to the DB2 database
  • string $password: the password that corresponds to the above username
  • string $database: the name of the database to from which health data will be obtained
  • string $port: the number of the port under which the database resides
  • string $language: the language code used to determine the description language
  • boolean $defset: the definition request flag used to determine whether or not to append health definitions
