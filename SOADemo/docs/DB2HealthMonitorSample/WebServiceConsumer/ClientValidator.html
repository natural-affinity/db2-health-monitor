Class ClientValidator

Description
Description | Methods (details) Constants (details)

A static class designed specifically for the Web Service Consumer package to perform validation.

Stores a set of regular expressions and paths to XML schemas in order to validate data from both the Web Service Consumer GUI and the Web Service Producer.
